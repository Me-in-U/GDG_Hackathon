## 앱의 용도
- 지역에서 달리기를 함께 즐기는 커뮤니티 앱
- 루트를 혼자 달리거나 여러 사람이 릴레이 형식으로 이어 달려 목표를 달성
- 목표 루트를 완주하면 거리 점수와 함께 해당 루트 모양의 뱃지 획득

## 개발 전 기대 기능

1. 달리기 루트를 만든 후 그 루드에 따라 같은 모양의 뱃지 지급 예상
![image](https://github.com/user-attachments/assets/493016ae-a645-4ed9-96da-6c27cbd2ab06)
<p align="center">
  <img src="https://github.com/user-attachments/assets/a6da533e-a6b9-4bc7-9434-aea3b0d829f0" alt="Image" width="80%" />
</p>

2. 함께 달리기
지역 이웃들과 함께 릴레이 형식으로 특정 경로 생성

3. 뱃지 수령
완주한 경로의 그림을 뱃지로 수령, 리워드로 저장

4. 사용자 커스텀 경로 생성
경치 좋은 경로, 카페 거리 등 주민 추천 경로와 커스텀 경로를 생성

5. 랭킹
달린 거리를 바탕으로 지역 주민들 사이에서 랭킹을 매김



## 기대효과
- 건강과 웰빙 -> 달리기를 통한 체력 향상 및 부가 이점, 개인의 건강 증진
- 지속 가능한 도시와 공동체 -> 다양한 사람들과 미션을 하며 증가된 공동체 정신, 경로에 위치한 지역 사회 활성화
- 파트너십 활성화 -> 지역 커뮤니티의 네트워크 강화, 이어달리기를 통한 파트너십 향상
